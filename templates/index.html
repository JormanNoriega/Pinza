Modelo matemático

Palanca de primer grado

Equilibrio ideal de momentos F_entrada * d1 = F_salida * d2
Ventaja mecánica usada en la app VM = F_salida / F_entrada = d1 / d2
Modelo extendido para pérdidas F_salida,real = (F_entrada * d1 / d2) * eta

La pinza se representa como dos brazos rígidos articulados en un fulcro central. Cada mango genera un momento sobre el pivote y ese momento se transmite a las mordazas.

Para que la experiencia sea educativa y ampliable, el cálculo distingue entre el caso ideal de equilibrio perfecto y un caso efectivo con pérdidas por fricción, amortiguamiento y flexibilidad.
